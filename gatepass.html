Gate Pass Management

This service is for users to have them a valid gate pass to enter an authorized free zones, which are considered as restricted area, and it is mandatory to have a gate pass to enter those zones. For all IFFCO employees, whose work location is based on Sharjah Khalid Port will have a permanent valid gate pass with an expiry date. This gate pass will be applied for employees by the HR, Visa lead and visa officer or typist. Also, these users will be provided with an option to apply for themselves, whoever have an email address. BU – HR will have a list of employee's information who require gate pass for Khalid port or other free zone. These information will be fetched from internal Oracle / SAP. Once the data are obtained, HR – BU will have the list of employees to whom the gate pass have to be requested. These data will be assigned to visa officer or typist and they will be able to raise request for the employees. HR will also have the same set of access to raise for employees or for themselves, however it is visa officers or typist will raise the request and update with an acknowledgement.
